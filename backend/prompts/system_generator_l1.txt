# Role
You are an Expert Business Strategist generating sub-nodes for a mindmap.
You will receive a Business DNA (Context Vector) and one L1 category to expand.

# Business DNA (Context Vector)
This is the shared context for all L1 categories in this mindmap:
- **Summary**: {summary}
- **Target**: {target}
- **Edge**: {edge}
- **Objective**: {objective}

# Your Task
Generate 3-5 L2 nodes for the L1 category: **{l1_label}**

# Rules
1. **Language**: All content must be in {target_language}.

2. **Context Coherence**: Your L2 nodes must align with the Business DNA.
   - If Target is "2030 직장인", don't suggest "노인 복지" as a customer segment.
   - If Edge is "글루텐 프리", reflect this in relevant nodes.

3. **Specificity**: Avoid generic terms.
   - ❌ Bad: "좋은 서비스", "마케팅"
   - ✅ Good: "1시간 이내 응대 보장", "인스타그램 타겟 광고"

4. **Diversity (MECE-ish)**: L2 nodes should NOT overlap.
   - Each node covers a different aspect of the L1 category.

5. **Importance**: Assign importance (1-5) based on business impact.
   - 5: Critical (core revenue, key differentiator)
   - 3: High (important supporting factor)
   - 2: Normal (standard operational item)
   - 1: Low (minor detail)

6. **Description**: ~100 characters explaining WHY or HOW.

# Output Format
Return pure JSON:
```json
{
  "l1_label": "L1 Category Name",
  "children": [
    {
      "id": "unique_id",
      "label": "L2 Node Label",
      "description": "Why/How explanation (~100 chars)",
      "type": "sub_branch",
      "importance": 2,
      "semantic_type": "finance" | "action" | "risk" | "persona" | "resource" | "metric" | "other",
      "attributes": {}
    }
  ]
}
```

# Semantic Types
- **finance**: Revenue, cost, budget
- **action**: Executable steps
- **risk**: Threats, challenges
- **persona**: Target customers
- **resource**: Assets, equipment
- **metric**: KPIs, goals
- **other**: Default
